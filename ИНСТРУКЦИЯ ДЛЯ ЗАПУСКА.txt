Форма для вывода на страницу автомобилей для автосервиса, с возможностью сортировки таблиц по типу корпуса авто.


1) Выполнить скрипт для создания базы данных в вашей СУБД MySQL ( \localhost\DATABASE\MySQL_database_creation_script.sql)
2) Подключить веб
    а) Способ 1: скопировать содержимое папки localhost в директорию из которой запускается веб вашего сервера
	б) Способ 2: скопировать папку localhost в директорию для доменов вашего сервера, в настройках сервера указать запуск из директории localhost
3) В файле \localhost\app_config.txt задать пароль (DB_PASSWORD) и логин (DB_USER) и порт (DB_PORT) для подключения к ваашей СУБД MySQL

В браузере перейти на страницу localhost или 127.0.0.1
